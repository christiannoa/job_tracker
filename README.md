# job_tracker

A full-stack web application for tracking job applications, built with Python Flask, MySQL, and a polished dark-themed UI.

1. Clone the repo:
git clone https://github.com/christiannoa/job_tracker.git

2. Install dependencies:
pip install -r requirements.txt

3. Set up MySQL and run:
schema.sql

4. Create a `.env` file with your DB credentials

5. Run the app:
flask --app app run

Features
Dashboard — Stats overview with application pipeline visualization
Companies — Track companies you're targeting with CRUD operations
Jobs — Log job listings with salary, type, and skill requirements
Applications — Monitor application status through the hiring pipeline
Contacts — Manage recruiter and employee network contacts
Job Match — Skills-based matching that ranks jobs by compatibility percentage

Tech Stack
•	Database: MySQL
•	Backend: Python 3 with Flask
•	Frontend: HTML, CSS (Bootstrap optional)
•	Version Control: GitHub
